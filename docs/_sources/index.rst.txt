========================================
Welcome to projectmaker's documentation!
========================================

projectmaker is a Python command line tool to make blank project frameworks.

* Setup the folder structure.
* Create and link basic files.
* Initiate git repositories and integrate basic submodules.
* Setup a basic Hello world and its linked blank code libraries.

The source code can be found on `GitHub <https://github.com/fabquenneville/projectmaker>`_
