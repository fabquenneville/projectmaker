Use cases¶

This is a tool used to make my new projects / modules / website file and folder structure:

  • Make new Python module / application

  • Make a new php website

See the Manual for more details on the options and the default options.

Make new Python module / application¶

This is a command used to make a new project called project0 with:

  • A sphinx documentation and a script to prepare releases

  • A template setup script

  • A folder and file structure

  • A git repository with the first commits done and pushed

./projectmaker.py -path:/home/username/workplace/ -name:project0 -owner:"John Doe" -license:gpl3 -language:python -git:git@github.com:johndoe/project0.git

Make a new php website¶

This is a command used to make a new site called site0 with:

  • A folder and file structure

  • A basic hello world page with liked libraries in css, javascript and php

  • The Font-Awesome submodule

  • A git repository with the first commits done and pushed

./projectmaker.py -path:/home/username/workplace/ -name:site0 -owner:"John Doe" -license:none -language:php -docs:none -readme:none -git:git@github.com:johndoe/site0.git
